
📘 C++ Projects – FEUP (Engenharia Eletrotécnica e de Computadores)

Este repositório contém exercícios e projetos em C++ desenvolvidos no âmbito das unidades curriculares de programação na FEUP – Faculdade de Engenharia da Universidade do Porto.

🎯 Objetivo

Este repositório tem como objetivo:

Consolidar bases de programação em C++
Praticar Programação Orientada a Objetos (OOP)
Implementar estruturas de dados clássicas
Desenvolver pensamento algorítmico
Preparação para testes e exames
📂 Conteúdo

O repositório inclui exercícios sobre:

✔ Vetores (vector)
✔ Strings
✔ Funções e modularização
✔ Classes e objetos
✔ Ponteiros e referências
✔ Listas, stacks e queues (std)
✔ Algoritmos de pesquisa e padrões
✔ Exercícios típicos de exames
🛠️ Tecnologias
C++
GCC / G++
Standard Library (STL)
VS Code / Code::Blocks / CLion

Linux / Mac
g++ ficheiro.cpp -o programa

programa.exe
